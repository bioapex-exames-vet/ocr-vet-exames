# ocr-vet-exames
Sistema web para leitura automática de exames veterinários (hemograma e bioquímico) impressos em papel térmico por equipamentos laboratoriais. Realiza upload da imagem, aplica OCR, extrai parâmetros clínicos, preenche template padrão com logo, gera PDF, salva no Google Drive e envia por e-mail, com login seguro.
